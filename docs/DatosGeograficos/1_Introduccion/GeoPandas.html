Tipos de datos geogr√°ficos¬∂

La informaci√≥n geogr√°fica generalmente se agrupa en dos grandes categor√≠as:

  • Datos r√°ster

  • Datos vectoriales

R√°ster: Los datos r√°ster son aquellos donde la informaci√≥n se encuentra codificada sobre una grilla o red, donde cada celda (o p√≠xel) presenta un valor num√©rico. Los datos r√°ster son usados com√∫nmente en im√°genes satelitales, en este caso cada p√≠xel representa un espacio definido por la granularidad o calidad de la imagen tomada, por ejemplo, un p√≠xel puede representar una superficie de 30 metros por 30 metros.

Sin embargo, en esta clase nos vamos a concentrar en los datos vectoriales. En los datos vectoriales contamos con los siguientes tipos de objetos b√°sicos: puntos, l√≠neas, pol√≠gonos; as√≠ tambi√©n tenemos multipunto, multil√≠nea y multipol√≠gono. La informaci√≥n se estructura a partir de las coordenadas de los v√©rtices de cada uno de esos objetos.

Introducci√≥n a GeoPandas¬∂

GeoPandas es la librer√≠a de Python por excelencia para el procesamiento de datos vectoriales. Provee f√°cil acceso a una serie de funcionalidades empleando diversas librer√≠as:

  • Manipulaci√≥n de datos con Pandas.

  • Lectura de archivos espaciales a trav√©s de Fiona.

  • Objetos vectoriales y operaciones vectoriales con Shapely.

  • Visualizaci√≥n de mapas con Descartes y Matplotlib.

Importando datos¬∂

La datos vectoriales com√∫nmente se presenta en archivos como el formato shapefile (.shp) de ESRI, archivos GeoJSON, bases de PostGIS (es una extensi√≥n del motor de bases de datos PostgreSQL), etc. Con la librer√≠a GeoPandas se pueden leer muchos de estos archivos, interfaceando con la librer√≠a GDAL/OGR, usando la funci√≥n geopandas.read_file. Adem√°s, tambi√©n es posible leer informaci√≥n geogr√°fica desde archivos planos (.csv) donde tenemos las coordenadas de puntos que referencian alg√∫n objeto. Por √∫ltimo, tambi√©n existen casos en los que la informaci√≥n est√° guardada en formato WKT (Well-Known Text).

def from_x_y(df, x, y, crs='EPSG:4326'):
    gdf = geopandas.GeoDataFrame(df.drop([x, y], axis=1), # eliminamos las columnas originales
                                crs=crs, # Agregamos CRS
                                geometry=geopandas.points_from_xy(df[x], df[y])) # junto "x" e "y" y lo paso a Point
    return gdf

Distancia a√©rea y centroide¬∂

Otro uso muy com√∫n en operaciones geogr√°ficas, especialmente cuando hacemos an√°lisis urbanos, es el c√°lculo de una distancia. Por simplicidad en este caso vamos a calcular la distancia a√©rea (o distancia euclidea).
